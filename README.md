# Daily Reflection Tree

This project implements a deterministic reflection tool based on 3 psychological axes:

Axis 1: Locus of Control
Axis 2: Contribution vs Entitlement
Axis 3: Radius of Concern

The tool guides employees through structured reflection using fixed options.

No LLM is used at runtime.

Files:
tree/reflection-tree.tsv → tree structure
tree/tree-diagram.md → visual diagram
write-up.md → design explanation

Optional agent can run using Python CLI.
